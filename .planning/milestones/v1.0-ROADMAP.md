# Milestone v1.0: MVP

**Status:** SHIPPED 2026-04-17
**Phases:** 1-5
**Total Plans:** 5

## Overview

AtomicBrain v1.0 MVP delivers the complete local-first knowledge management pipeline: from raw file ingestion through AI-powered concept extraction and hierarchical organization, to agent-accessible search via MCP and a human-friendly web dashboard. All five phases shipped in 3 days (Apr 15–17, 2026).

## Milestone Summary

**Key Decisions:**
- Use MarkItDown as primary ingestion tool, Docling as high-fidelity fallback — avoids heavy OCR for simple files
- Manual `FastEmbedGenerator` wrapper (not LanceDB's built-in) — ensures robust local embedding without ONNX version conflicts
- Lazy agent imports in CLI — allows MCP and search to run without LLM API key configured
- FastAPI serves static frontend files — single `atomic-brain dashboard` command, no separate frontend server needed
- Pure file system vault (no DB for Markdown) — seamless Obsidian compatibility, zero vendor lock-in

**Issues Resolved:**
- LanceDB embedding integration required manual wrapper due to FastEmbed version constraints
- FTS (full-text search) registration needed explicit enable during table creation

**Issues Deferred:**
- Email ingestion (.eml/.msg) — deferred to v2 (ING-03)
- Fractal clustering for 100K+ concepts — deferred to v2 (ORG-03)
- Agent write access via MCP — deferred to v2 (MCP-03)
- Interactive graph visualization — deferred to v2 (UI-03)

**Technical Debt Incurred:**
- Tests exist but are mostly unit tests with mocks; integration tests against real LanceDB not yet written
- Frontend has no tests
- `OPENAI_API_KEY` required for organization agents (LiteLLM not yet fully wired for local Ollama)
