# REQUIREMENTS ARCHIVE — AtomicBrain v1.0 MVP

**Archived:** 2026-04-17
**Milestone:** v1.0 MVP (Phases 1–5)
**Status:** All v1 requirements shipped and validated.

---

## v1 Requirements (MVP) — COMPLETED

### Ingestion & Processing
- [x] **ING-01**: High-fidelity ingestion of text, Markdown, PDF, and DOCX using **MarkItDown** (primary) and **Docling** (high-fidelity fallback) for conversion. — ✓ Phase 1
- [x] **ING-02**: Atomic extraction of content into "Concepts" with standardized Markdown metadata. — ✓ Phase 1

### Search & Retrieval
- [x] **SRCH-01**: Hybrid search functionality combining **Vector Search** (LanceDB/FastEmbed) and **Keyword Search** (FTS). — ✓ Phase 2
- [x] **SRCH-02**: Semantic retrieval that grounds search results in the original source documents (source path + concept ID in results). — ✓ Phase 2

### The Brain: Organization
- [x] **ORG-01**: **Autonomous Concept Extraction**: AI identifies core ideas and relationships using Pydantic AI `RefinementAgent`. — ✓ Phase 3
- [x] **ORG-02**: **Hierarchical Vault**: Concepts organized into multi-level folder structure via `VaultManager` + `OrganizationAgent`. — ✓ Phase 3

### Agent Bridge: MCP
- [x] **MCP-01**: **Read-only MCP Server**: `search_concepts` and `get_concept_details` tools for external AI agents. — ✓ Phase 4
- [x] **MCP-02**: **Dynamic Tool Registration**: Server registers `query_<category>` tools based on vault folder scan. — ✓ Phase 4

### Human Interface
- [x] **UI-01**: **Web UI Dashboard**: Local dashboard with status, search, and concept viewer at `atomic-brain dashboard`. — ✓ Phase 5
- [x] **UI-02**: **Obsidian Native**: Vault stored as pure Markdown files with YAML frontmatter in `vault/Concepts/`. — ✓ Phase 1

## v2 Requirements (Deferred — carry forward)

- [ ] **ING-03**: Support for complex formats: Emails (.eml/.msg), OCR for images, and audio transcription.
- [ ] **ORG-03**: **Fractal Clustering**: Deep recursive organization for 100K+ concepts with nested sub-topics.
- [ ] **MCP-03**: **Agent Write Access**: Authorization flow for external agents to create/edit concepts.
- [ ] **UI-03**: **Interactive Graph Explorer**: Visual representation of the hierarchical knowledge structure.

## Out of Scope

- **Cloud Sync**: AtomicBrain is strictly local-first. Still valid.
- **Social Features**: Personal brain, not collaborative wiki. Still valid.
